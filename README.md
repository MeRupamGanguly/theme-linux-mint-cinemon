## ROFI 

```bash

mkdir /home/rupx/.local/share/rofi
cd /home/rupx/.local/share/rofi
nano rofi-find.sh

find /home/* -type f | rofi -dmenu -i -p "Find File:" | xargs -I {} xdg-open {}


chmod +x rofi-find.sh


cd ~
mkdir /home/rupx/.config/rofi
cd /home/rupx/.config/rofi
```
